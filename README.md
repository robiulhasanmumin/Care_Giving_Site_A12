# üè† Care.People - Compassionate Caregiving Services

**Care.People** is a modern, responsive full-stack web application built with **Next.js 15** and **MongoDB**. It is designed to connect families with professional caregivers providing services such as Baby Sitting, Elderly Care, and Medical Support.

---

## ‚ú® Key Features

* **Dynamic Service Filtering:** Interactive category-based filtering (e.g., Baby Care, Elderly Service) on the home page for a seamless user experience.
* **Detailed Service Pages:** Dynamic routing used to display comprehensive information for each specific service.
* **Dynamic Metadata:** Optimized for SEO; each service page generates its own unique title and description based on database content.
* **Specialized Booking System:** Integrated booking functionality with real-time price calculation based on service duration.
* **Responsive UI/UX:** Fully optimized for all screen sizes (Mobile, Tablet, Desktop) using **Tailwind CSS** and **DaisyUI**.
* **Next.js Server Actions:** Secure and efficient data fetching and database operations directly from the server.
