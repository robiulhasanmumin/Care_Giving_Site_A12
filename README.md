# 🏠 Care.People - Compassionate Caregiving Services

**Care.People** is a modern, responsive full-stack web application built with **Next.js 15** and **MongoDB**. It is designed to connect families with professional caregivers providing services such as Baby Sitting, Elderly Care, and Medical Support.

### 🚀 Live Links
- **Live URL:** [https://care-giving-site.vercel.app/](https://care-giving-site.vercel.app/)

## 📸 Project Overview
A specialized platform where users can browse, filter, and book professional caregiving services with a seamless user experience.

## ✨ Key Features

* **Dynamic Service Filtering:** Interactive category-based filtering (e.g., Baby Care, Elderly Service) for a seamless user experience.
* **Dynamic SEO Metadata:** Optimized for search engines; each service page generates its own unique title and description dynamically based on database content.
* **Specialized Booking System:** Integrated booking functionality with real-time price calculation based on the duration of service.
* **Detailed Service Pages:** Uses **Dynamic Routing** to display comprehensive information for each specific caregiving service.
* **Next.js Server Actions:** Secure and lightning-fast data fetching and database operations handled directly on the server.
* **Responsive UI/UX:** Fully optimized for all screen sizes (Mobile, Tablet, Desktop) using **Tailwind CSS** and **DaisyUI**.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Authentication:**	NextAuth.js
- **Database:** [MongoDB (Native Driver)](https://www.mongodb.com/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
- **Server Logic:** Next.js Server Actions
- **Icons:** React Icons
